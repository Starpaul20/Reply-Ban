## INFORMATION ##

Reply Ban (1.3) for MyBB 1.8
Created by: Starpaul20
Copyright: ©2015
License: GPL

Allows you to ban users from replying to a thread.

The reply ban page can be accessed by clicking Reply Bans link located next to Thread Modes on top of threads. This plugin offers full language support.


## INSTALLATION ##

1. Upload all files above, keeping the file structure intact.
2. Go to Configuration > Plugins
3. Click "Install & Activate"
4. Enjoy!


## UPDATING ##

If you're updating from any previous version, you must first deactivate the plugin (do not uninstall), upload all new files and reactivate.
